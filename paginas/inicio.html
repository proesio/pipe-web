Bienvenido al ORM PIPE
Fácil de Instalar

Debido a que el ORM PIPE esta pensado para utilizarse en proyectos que no siguen la estructura de un framework, tienes toda la libertad de pegar el directorio PROES que contiene la clase PIPE.php y el archivo de configuración de la base de datos PIPE_CONEXION_BD.php en cualquier parte de tu proyecto.

Fácil de Configurar

PIPE soporta 5 controladores de bases de datos. mysql, pgsql, sqlite, oci y sqlsrv.

Para configurar la conexión a nuestra base de datos, abrimos y editamos el archivo PIPE_CONEXION_BD.php que se encuentra en el directorio PROES.

<?php
	const BD_CONTROLADOR = 'controlador';
	const BD_HOST = 'host';
	const BD_PUERTO = 'puerto';
	const BD_USUARIO = 'usuario';
	const BD_CONTRASENA = 'contraseña';
	const BD_BASEDATOS = 'basedatos';
	require_once "Conexion.php";
	require_once "PIPE.php";

Para utilizar PIPE en nuestro código de programación incluimos el archivo PIPE_CONEXION_BD.php y usamos el namespace PROES\PIPE.

<?php
	require_once "PROES/PIPE_CONEXION_BD.php";
	use PROES\PIPE;
Fácil de Usar

Creación de un Modelo.

<?php
	use PROES\PIPE;
	class Usuario extends PIPE {
		// Todo tu código aquí.
	}

Vamos a obtener todos los datos de la tabla usuarios por medio del modelo.

<?php
	require_once "PROES/PIPE_CONEXION_BD.php";
	include "Usuario.php"; // Incluimos el archivo Usuario.php que contiene la clase Usuario.
		$usuarios = Usuario::todo();

Ahora, vamos a obtener todos los datos de la tabla usuarios por medio del Constructor de Consultas PIPE.

<?php
	require_once "PROES/PIPE_CONEXION_BD.php";
	use PROES\PIPE;
		$usuarios = PIPE::tabla('usuarios')->todo();
