Bienvenido al ORM PIPE
Fácil de Instalar

Debido a que el ORM PIPE está pensado para utilizarse en proyectos que no siguen la estructura de un framework, tienes toda la libertad de pegar el directorio PIPE en cualquier parte de tu proyecto.

Fácil de Configurar

PIPE es muy sencillo de configurar.

PIPE soporta 5 controladores de bases de datos. mysql, pgsql, sqlite, oci y sqlsrv.

PIPE soporta 2 idiomas para mostrar errores SQL y mensajes del ORM PIPE. es, en.

Para configurar la conexión a nuestra base de datos, abrimos y editamos el archivo basedatos.php que se encuentra en la siguiente ruta PIPE/Config/basedatos.php.

<?php
	const BD_CONTROLADOR = 'controlador'; // Soporta mysql, pgsql, sqlite, oci y sqlsrv.
	const BD_HOST = 'host';
	const BD_PUERTO = 'puerto';
	const BD_USUARIO = 'usuario';
	const BD_CONTRASENA = 'contraseña';
	const BD_BASEDATOS = 'basedatos';
	const BD_CODIFICACION = 'utf8';

Para configurar el idioma en el cual se muestran los errores SQL y mensajes del ORM PIPE, abrimos y editamos el archivo idioma.php que se encuentra en la siguiente ruta PIPE/Config/idioma.php.

<?php
	const IDIOMA = 'idioma'; // Soporta es, en.

Para utilizar PIPE en nuestro código de programación incluimos el archivo PIPE.php que se encuentra en la siguiente ruta PIPE/PIPE.php.

<?php
	require_once "PIPE/PIPE.php";
	// Todo tu código aquí.
Fácil de Usar

Creación de un Modelo.

PIPE por defecto tiene un directorio específico para guardar los modelos que serán importados en nuestro código automáticamente. Creamos el archivo Usuario.php y lo guardamos en el directorio PIPE/Modelos/Usuario.php.

<?php
	use PIPE\Clases\Modelo;
	class Usuario extends Modelo {
		// Todo tu código aquí.
	}

Vamos a obtener todos los datos de la tabla usuarios por medio del modelo.

<?php
	require_once "PIPE/PIPE.php";
	$usuarios = Usuario::todo();

Ahora, vamos a obtener todos los datos de la tabla usuarios por medio del Constructor de Consultas PIPE.

<?php
	require_once "PIPE/PIPE.php";
	$usuarios = PIPE::tabla('usuarios')->todo();
