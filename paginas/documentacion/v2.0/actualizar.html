Actualizar

PIPE tiene dos formas de actualizar datos en las tablas de la base de datos.

Forma #1 de actualización de datos por medio del modelo.

<?php
	require_once "PIPE/PIPE.php";
	// El método encontrar() recibe como parámetro el id de la llave primaria.
	$usuario = Usuario::encontrar(1);
	$usuario->nombres = 'John';
	$usuario->apellidos = 'Doe';
	$usuario->actualizar(); // Este método devuelve el número de registros actualizados.

Forma #2 de actualización de datos por medio del modelo.

<?php
	require_once "PIPE/PIPE.php";
	$usuario = Usuario::donde('id=? o id=?',[1,2])->actualizar(['nombres'=>'John','apellidos'=>'Doe']);
	// Si deseamos realizar una actualización masiva lo hacemos de la siguiente manera.
	$usuario = Usuario::instanciar()->actualizar(['nombres'=>'John','apellidos'=>'Doe']);

Forma #1 de actualización de datos por medio del constructor de consultas PIPE.

<?php
	require_once "PIPE/PIPE.php";
	// Si nuestra llave primaria no sigue la convención del ORM PIPE podemos especificarla de la siguiente manera.
	$usuario = PIPE::tabla('usuarios')->encontrar(1,'codigo');
	$usuario->nombres = 'John';
	$usuario->apellidos = 'Doe';
	$usuario->actualizar(); // Este método devuelve el número de registros actualizados.

Forma #2 de actualización de datos por medio del constructor de consultas PIPE.

<?php
	require_once "PIPE/PIPE.php";
	$usuario = PIPE::tabla('usuarios')->donde('id=? o id=?',[1,2])->actualizar(['nombres'=>'John','apellidos'=>'Doe']);
	// Si deseamos realizar una actualización masiva lo hacemos de la siguiente manera.
	$usuario = PIPE::tabla('usuarios')->actualizar(['nombres'=>'John','apellidos'=>'Doe']);

El método actualizarOInsertar() permite actualizar o insertar un nuevo registro en la base de datos.

<?php
	require_once "PIPE/PIPE.php";
	/*
		El primer parámetro hace referencia a los campos que serán actualizados.
		El segundo parámetro hace referencia a los campos que serán insertados en caso de no encontrar registros.
	*/
	$usuario = Usuario::donde('id=? o id=?',[1,2])->actualizarOInsertar(['nombres'=>'John','apellidos'=>'Doe'],['nombres'=>'John','apellidos'=>'Doe']);

Actualización de datos por medio del método consulta().

<?php
	require_once "PIPE/PIPE.php";
	// Este método devuelve el número de registros actualizados.
	$usuario = PIPE::consulta("actualizar usuarios asignar nombres='John',apellidos='Doe' donde id=1");

Actualización de datos por medio del método consultaNativa().

<?php
	require_once "PIPE/PIPE.php";
	// Este método devuelve el número de registros actualizados.
	$usuario = PIPE::consultaNativa("update usuarios set nombres='John',apellidos='Doe' where id=1");

Si utilizas el Modelo para interactuar con los datos de la base de datos, tendrás la ventaja de poder omitir las convenciones que tiene el ORM PIPE y adaptarse a la estructura que tenga tu tabla en la base de datos, ya que el Constructor de Consultas se adapta por defecto a las convenciones.
