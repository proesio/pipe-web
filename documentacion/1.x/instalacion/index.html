Instalación

PIPE es muy sencillo de instalar.

Debido a que el ORM PIPE esta pensado para utilizarse en proyectos que no siguen la estructura de un framework, tienes toda la libertad de pegar el directorio PROES que contiene la clase PIPE.php y el archivo de configuración de la base de datos PIPE_CONEXION_BD.php en cualquier parte de tu proyecto.

Serás tú el que decide el flujo y la estructura de tu aplicación web.
