Configuración

PIPE es muy sencillo de configurar.

PIPE soporta 5 controladores de bases de datos. mysql, pgsql, sqlite, oci y sqlsrv.

Para configurar la conexión a nuestra base de datos, abrimos y editamos el archivo PIPE_CONEXION_BD.php que se encuentra en el directorio PROES.

<?php
	const BD_CONTROLADOR = 'controlador';
	const BD_HOST = 'host';
	const BD_PUERTO = 'puerto';
	const BD_USUARIO = 'usuario';
	const BD_CONTRASENA = 'contraseña';
	const BD_BASEDATOS = 'basedatos';
	require_once "Conexion.php";
	require_once "PIPE.php";

Para utilizar PIPE en nuestro código de programación incluimos el archivo PIPE_CONEXION_BD.php y usamos el namespace PROES\PIPE.

<?php
	require_once "PROES/PIPE_CONEXION_BD.php";
	use PROES\PIPE;
