Configuración

PIPE es muy sencillo de configurar.

Si hemos instalado PIPE por medio de composer debemos incluir el archivo vendor/autoload.php en nuestro código de programación para poder utilizar el ORM PIPE.

Si por el contrario hemos instalado PIPE manualmente debemos incluir el archivo PIPE.php que se encuentra en la raíz del código fuente descargado.

Para configurar el ORM PIPE inicializamos la configuración de la siguiente manera.

<?php
	require "vendor/autoload.php";
	
	use PIPE\Clases\PIPE;
	use PIPE\Clases\Configuracion;
	
	Configuracion::inicializar([
		'BD_CONTROLADOR' => 'controlador', // Soporta mysql, pgsql, sqlite y sqlsrv.
		'BD_HOST' => 'host', // IP o DNS del servidor.
		'BD_PUERTO' => 'puerto', // Puerto donde se ejecuta el servicio.
		'BD_USUARIO' => 'usuario', // Usuario de la base de datos.
		'BD_CONTRASENA' => 'contrasena', // Contraseña de la base de datos.
		'BD_BASEDATOS' => 'basedatos', // Nombre de la base de datos.
		'BD_CODIFICACION' => 'codificación', // Juego de caracteres. Ej: utf8.
		'IDIOMA' => 'idioma', // Idioma para mostrar errores SQL y mensajes del ORM PIPE. Soporta es, en.
		'RUTA_MODELOS' => 'ruta/completa/a/mis/modelos', // Ej: C:\xampp\htdocs\mi-proyecto\modelos.
		'ZONA_HORARIA' => 'Zona/Horaria' // [Valor UTC por defecto] Ej: America/Bogota.
	]);
	
		/*
			Finalmente verificamos que hemos inicializado la configuración correctamente
			imprimiendo la versión del ORM PIPE.
		*/
	
	echo PIPE::VERSION;
